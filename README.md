Zch，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库

## 介绍

这是我在学习 `vue` 过程中做的一个 UI 框架，希望对你有用。

1. 安装

```
$ npm install zch-gulu
$ yarn add zch-gulu
```

2. 使用

```
import { ZchButton } from 'zch-gulu'
import 'zch-gulu/dist/index.css'

<zch-button>Button</zch-button>
```
